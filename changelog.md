# 📝 Changelog
All notable changes to the Crispy Hub website are documented here.

## [1.2.0] – 2026-06-20
### ✅ Added
- *Interactive Lightbox Gallery*: Added Lightbox2 library to index.html, menu.html, and promotions.html; updated all images to work with popup view
- *Interactive Branch Map*: Integrated Leaflet.js into location.html with markers for all South African branches
- *Form Validation*: Added client‑side validation for contactus.html and enquiries.html using JavaScript
- *SEO Files*: Created robots.txt and sitemap.xml to meet Part 3 SEO requirements
- *Full Referencing*: Added Harvard‑style references in:
  - <head> of every HTML page
  - Top of myjavascripts.js
  - README.md and Changelog.md
- *Responsive Navigation*: Fixed mobile menu layout based on Part 2 feedback
- *Alt Text*: Added descriptive alt attributes to all images for accessibility & SEO

### ✅ Improved
- Updated meta descriptions, keywords, and titles for all pages
- Cleaned up and organised JavaScript code into clear sections
- Added error messages and success feedback for form submissions
- Optimised file paths and removed unused code


## [1.1.0] – 2026-06-19
### ✅ Added
- Created detailed README.md file documenting features, tech stack, and structure
- Implemented dynamic menu generation using JavaScript arrays

- Integrated jQuery v3.7.1 across all pages

### ✅ Changed
- Corrected script loading order: jQuery loads before custom JavaScript

- Improved responsive layout for better mobile display

### ✅ Fixed

- Removed unused styles and cleaned up code
