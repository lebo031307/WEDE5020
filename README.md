## 📋 Project Overview
Crispy Hub is a fully responsive multi‑page website built for a South African fast‑food restaurant. It meets all requirements for Part 2 and Part 3 of the WEDE5020 assessment, including interactivity, SEO, forms, and proper documentation.

---

## ✨ Key Features
✅ *Responsive Design*: Works on mobile, tablet, and desktop using CSS Flexbox
✅ *Interactive Menu*: Dynamic menu generation + real‑time search filtering
✅ *Image Gallery / Lightbox*: Click any food image to open a larger view
✅ *Interactive Branch Map*: Leaflet.js map showing all South African locations
✅ *Functional Forms*:
  - contactus.html: General contact form with validation
  - enquiries.html: Service/product enquiry form with validation
✅ *SEO Optimised*:
  - Unique page titles, meta descriptions, and keywords
  - robots.txt and sitemap.xml for search engines
  - Alt text for all images
✅ *Clean, Valid Code*: Semantic HTML5, modular CSS, and organised JavaScript

---

## 🛠️ Built With
- *HTML5*: Semantic structure, forms, and accessibility
- *CSS3*: Flexbox layout, responsive styling, and custom design
- *JavaScript (ES6)*: Dynamic content, search, form validation, and interactivity
- *jQuery v3.7.1*: DOM handling and library compatibility
- *Leaflet.js v1.9.4*: Interactive map functionality
- *Lightbox2 v2.11.4*: Image gallery popup
- *Font Awesome v6.5.0*: Icons and UI elements
- *Google Fonts (Poppins)*: Typography
