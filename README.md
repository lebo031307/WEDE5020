Web Development Project
## Project Title
Crispy Hub
CrispyHub is a modern, responsive multi-page web application built for a fast-food restaurant specializing in delicious fried chicken, burger meals, combo boxes, and desserts. This project showcases structured semantic HTML, a robust layout design using CSS Flexbox, interactive components powered by JavaScript, and library optimization via jQuery.

## 🚀 Key Features

- *Dynamic Interactive Menu:* Generates responsive food product cards dynamically from an internal data array using JavaScript DOM injection.
- *Real-Time Live Search:* Users can filter menu options instantly by typing names into a search box to find items immediately.
- *Advanced Flexbox Grid Layout:* A custom-engineered CSS layout that aligns items perfectly side-by-side using responsive wrapper rows while completely eliminating default listing bullet points.
- *Client Enquiry Validation:* Secure validation handling on the contact form to ensure proper user submission formatting.
- *SEO Optimized:* Structured search keywords and definitions configured safely within page metadata across all assets.

## 🛠️ Built With

- *HTML5:* Semantic structure (<header>, <nav>, <main>, <section>, <footer>).
- *CSS3:* Custom styles utilizing structural Flexbox rules (display: flex, flex-wrap: wrap) and clean spacing variables.
- *JavaScript (ES6):* Dynamic card generation, filtering array loops (.forEach), and interactive content manipulation.
- *jQuery Library:* Production CDN integration to accelerate DOM rendering and script safety.
